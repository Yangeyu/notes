##光标移动

##实用命令
| 快捷键               | 说明                                          |
|----------------------|-----------------------------------------------|
| `:g/^\s*$/d`         | 删除空行                                      |
| `:g/test/d`          | 删除所有包含test的行                          |
| `:v/test/d`          | 删除所有不包含test的行                        |
| `:%s/^/test/`        | 在行首加入test                                |
| `:%s/$/test/`        | 在行尾加入test                                |
| `:%s/p1/p2/gc`       | 替换当前文件中的p1为p2,并且每处询问你是否替换 |
| `:g/^\(.\+\)$\n\1/d` | 去除重复行(先排序)                            |
| `:%s/.\{10}//`       | 删除每行前10个字符                            |
| `vim -u NONE -N`     | 开启vim时不加载vimrc文件                      |
| `gf`                 | 跳转到光标下的文件名                          |
